Blog structure:
/bikepacking/ - only bikepacking and travel related posts
/tech/ - only technical writing
/blog/ - combination of the other two sub blogs

Ensure that whenever a post is added the relevant page and xml feeds are updated, this will generally be 2: blog AND (bikepacking OR tech)

Do not use flexbox unless absolutely necessary, always check if you plan on using it.

Always prefer to directly edit files rather than using sed to make sweeping changes.

When modifying styles.css, introducing new HTML patterns, or making widespread changes to page structure update STYLE_GUIDE.md to reflect the change.
