About

This website is a place to put some of the things I'm working on without it sitting on a social media platform.

In the interest of using the simplest tools for the job, and in the spirit of the indie web, I've used basic html and css with no framework or even any javascript. For a bit of fun I've chosen to also take aesthetic inspiration from the early 2000s/late 90s.

Inspirations:

  • George Alex Grant's homepage
  • geocities.org (archive)
  • Udi Finkelstein's homepage - found on the geocities archive, it's where I nicked the nostalgic background image from.
  • gifcities.org
