human.json

Published March 18th 2026

I've added a human.json file to my site — it's new age thing that fits in well with the philosophy around rewilding the web. It allows humans to vouch for each other's web footprints which is something that could become increasingly useful in the world of openclaw.

For more information see https://codeberg.org/robida/human.json.

If you're a human I know and would like a vouch, let me know.

Thanks to Evan Hahn for making me aware of it.
